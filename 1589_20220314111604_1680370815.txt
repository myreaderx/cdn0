1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\">投资者热衷于把元宇宙描述为开发空间有限的单一的元宇宙，元宇宙的不动产因此具有稀缺性并能够保值。但也有人认为这是个拉盘抛售计谋，设计者低位投入资金然后炒热概念拉涨，再在高位抛售割韭菜。因为在互联网的意义上，并不存在单一的元宇宙。Meta的Horizon世界和微软的Mesh之间并不能互联互通，他们只是独立的VR（虚拟现实）应用程序。</span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\"> </span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\">现在对元宇宙描述的奇怪之处在于，如果某公司说他们的VR应用程序、电子游戏，抑或是社交平台是“元宇宙的一部分”，只因为元宇宙是未来，那他们的产品也必然是未来。就好比说AR（增强现实）是未来，然后谷歌眼镜是AR产品，所以谷歌眼镜就是未来。</span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\"> </span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\">元宇宙的房产热也是在这种叙述框架下出现的。虚拟地产公司Metaverse Group的负责人甚至把在元宇宙平台购买土地，比做是在纽约曼哈顿大开发地价飞涨之前的圈地。</span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\"> </span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\">更确切地说，Decentraland或者Sandbox之类的平台出售虚拟世界中基于非同质化代币（NFT）的资产，但是这些虚拟空间相互之间并没有交集。网络研究专家丹·奥尔森（Dan Olson）指出，购买元宇宙的地产实际上买到的是那个平台提供的服务。而一旦平台的运营出现问题，投资就会血本无归。</span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\"> </span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\">前面之前提到的把元宇宙地产比做在曼哈顿买房，但实际上任何人都可以在元宇宙创造出无数个触手可及的曼哈顿。这意味着用户购买这个平台的曼哈顿而不是那个平台的曼哈顿的唯一原因，只是这个平台提供的服务比别人好。</span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\"> </span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\">Tokens.com（该公司持有Metaverse Group 50%的股份）说他们在Decentraland平台的一座塔楼已经破土动工，这个描述就好像在说现实生活中的房地产一样。然而不要忘记它实际描述的是设计3D模型或者虚拟环境。软件工程师史蒂芬·迪尔（Stephen Diehl）认为，这种讲述更多的是构筑了一个故事，而不是描绘一个技术过程。人们需要这种故事，因为归根结底他们买的只是电脑里的一些数字而已。</span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\"> </span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\">用户在Decentraland中买卖土地使用的货币并不是美元，大部分地块甚至不能用流行的比特币去购买。平台有自己的加密货币，叫做mana， 技术上是ERC-20代币。这也意味着尽管Decentraland使用以太坊区块链构建，但mana的币值可能比以太币更不稳定。</span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\"> </span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\">去年十二月下旬，Decentraland上最便宜的地块售价在4千mana左右，大约相当于1万5千美金。假设你买了一块地皮，你就持有了这块地皮，直到下一个买家买走为止，因为非同质化代币mana不同于比特币，它是不能互换的。但另一方面，如果一个用户持有大量mana，他可以把mana出售，包括卖给那些想要买地皮的人。而mana的市场很小，地皮又那么贵，因此地价也很容易大幅波动。</span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\"> </span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\">实际上mana的币值暴涨的情况已经发生过几次。在脸书更名为Meta之后，Mana币值从之前很少突破1美元突然飙升至3.71美元。十一月下旬，第116号地块以61万8千mana价格售出，被称为史上最大的元宇宙土地收购，彼时mana价格约为4.1美元。两天后又上涨到了5.79美元。短短一个月内币值翻了五倍多。加密货币价值是否是个拉高套现的计划，不免让人怀疑。在mana币值创下历史新高的时候，该货币的交易量也达到了惊人的114亿美元，让地块售价的240万美元显得有点微不足道。</span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\"> </span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\">这些元宇宙平台可能是互联网的未来，也可能不是，但大量的资金已经投入了平台里，一旦破灭就会有人被套牢，这些人也许一开始只是被一个好故事套住了。奥尔森认为，这些故事瞄准的对象也许是脆弱的中产阶级，那些感觉自己正被系统挤压，偿付能力逐渐失控的人。平台向他们推销说这是一个机会，你只需要在正确的时间押注正确的风口，就能套现离场。</span></p><p style=\"max-width: 100%;min-height: 1em;white-space: normal;line-height: 2em;margin-left: 16px;margin-right: 16px;overflow-wrap: break-word !important;box-sizing: border-box !important;\"><span style=\"color: rgb(60, 64, 67);font-size: 16px;\"><br></span></p><p style=\"margin-right: 16px;margin-left: 16px;outline: 0px;font-family: -apple-system, \" system-ui neue sc sans gb yahei ui arial sans-serif normal rgb><br
