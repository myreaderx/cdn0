\n<p>To get started, open your Stripe dashboard, go to Developers &gt; API Keys &gt; Created restricted API key.</p> \n<p>Give your key a descriptive name, choose the <code class=\"language-text\">Write</code> permission under <code class=\"language-text\">Checkout Sessions</code> and click <code class=\"language-text\">Create key</code>.</p> \n<p>Next, make a copy of Drive. Go to Tools &gt; Script Editor and replace the <code class=\"language-text\">Stripe API Key</code> with the key generated in the previous step.
