235);font-size: 17px;\"><strong><br></strong></span></p><p style=\"margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">一段时间以来，手机浏览器野蛮生长，违规从事互联网新闻信息服务，成为“自媒体”传播乱象的聚集地和放大器。此次专项集中整治和督导整改，把影响力较大的8款手机浏览器纳入首批名单进行重点集中整治。<br style=\"font-size: 16px;font-family: \" hiragino sans gb yahei sans-serif start normal 16px;\">据了解，此次集中整治将多措并举、标本兼治，着力解决三大突出问题：一是发布“自媒体”违规采编的各类互联网新闻信息，如歪曲解读经济民生政策、散布“小道消息”、传播谣言信息、翻炒旧闻编造“新闻”等；二是发布“标题党”文章，如恶意浮夸、“唱衰”“卖惨”、冒名炒作等；三是发布违背社会主义核心价值观的不良信息，如传播低俗图文视频、炒作明星绯闻隐私和娱乐八卦等。<br 16px;\">“近年来，在互联网上大多企业都在追求流量经济，纯粹的以流量、点击率作为追求的目标。其中，就包括一些搜索引擎、浏览器的经营者、开发设计者，他们以吸人眼球的方式追求流量经济，也以此种方式作为自己商业模式的核心。”亚太网络法律研究中心创始人、主任，北京师范大学法学院教授刘德良说。<br style=\"font-size: 16px;font-family: \" hiragino sans gb yahei sans-serif start normal rgb><br></span></p><p style=\"margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">刘德良说，在这种情况下，一些标题党文章、低俗新闻、黄色淫秽信息的弹窗广告等内容通过浏览器发布，从而达到追求经济利益的目的。这种方式在近几年来越来越普遍，严重扰乱了网络的传播秩序。但这种以污染网络生态环境作为盈利方式的行为引起了社会和公众的极大不满，在这种背景下，网信办开展了此专项整治活动。<br style=\"font-size: 16px;font-family: \" hiragino sans gb yahei sans-serif start normal rgb><br></span></p><p style=\"margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">北京市盈科律师事务所高级合伙人韩英伟律师表示，互联网散播的低俗网络信息给手机用户造成了严重的负面影响，甚至侵犯公民个人隐私等合法权益。为加强对公民信息保护，提高政府监管力度，保障社会健康网络信息文化发展，必须对此网络乱象进行集中整治。<br style=\"font-size: 16px;font-family: \" hiragino sans gb yahei sans-serif start normal rgb><br></span></p><p style=\"margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">在陕西师范大学副教授、网络与新媒体系主任郭栋看来，现在移动互联网空间确实存在着一些问题，其中浏览器问题最为明显。因为浏览器有着一个非常重要的特点：可以自动推送文章，但往往其推送的文章信源不权威，甚至有些文章未经过审核和把关便与读者见面，引起了用户群体中一些思想上、认识上的混乱。<br style=\"font-size: 16px;font-family: \" hiragino sans gb yahei sans-serif start normal rgb><br></span></p><p style=\"margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 17px;\"><strong><span style=\"font-size: 17px;color: rgb(0, 165, 235);\">加强网络平台监管<br>规范内容审核机制</span></strong></span></p><p style=\"margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 17px;\"><strong><span style=\"font-size: 17px;color: rgb(0, 165, 235);\"><br></span></strong></span></p><p style=\"margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">据介绍，此次集中整治和专项督导对手机浏览器提出了明确整改要求和具体标准，其中包括，不得发布“自媒体”违规采编的互联网新闻信息，不得PUSH弹窗“自媒体”发布的各类信息，不得使用断章取义、虚假夸大、攻击侮辱、耸人听闻等噱头式标题炒作热点敏感话题，不得发布无中生有、旧闻翻炒、拼凑剪接、捕风捉影等不实信息，不得发布低俗、血腥等不良信息，等等。<br style=\"font-size: 16px;\">同时，国家网信办有关负责人强调，10月27日至11月9日，各手机浏览器要对照问题清单深入开展自查整改，举一反三全面清理违规信息、严管“自媒体”账号。同时，建立完善总编辑负责制、内容审核管理规范，从制度层面采取有力有效措施，切实防止扰乱网络传播秩序问题的发生。<br style=\"font-size: 16px;font-family: \" hiragino sans gb style=\"margin-left: 8px;margin-right: 8px;\"><br></section><section style=\"margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">据了解，在自查整改结束后，网信部门将对自查整改情况进行检查评估，对整改后问题依然突出的手机浏览器，将依法依规进行严肃处置，直至取缔相关业务。此外，各地网信部门还将对属地手机浏览器进行全面摸底排查，一并按要求纳入专项整治和督导整改。<br style=\"font-size: 16px;font-family: \" hiragino sans gb yahei sans-serif 16px;\">对此，韩英伟说，此次整治加强了对各类网络平台有效监管，规范了内容审核管理机制等，促进网络传播秩序实现良性转变，为经济社会发展营造了良好的网上舆论氛围。对企业来讲，避免进行违规互联网信息服务，增强政府相关部门审核，切实履行社会责任；对公民来讲，促使网络传播健康、积极向上的文化信息成为网络公民的集体共识；对政府来讲，加强对网络违规信息的监管力度，与企业、公民共同维护和谐网络秩序。<br style=\"font-size: 16px;font-family: \" hiragino sans gb yahei sans-serif start normal rgb><br></span></section><section style=\"margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;font-family: \" hiragino sans gb yahei sans-serif start normal rgb><br></span></section><section style=\"margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: style=\"font-size: 16px;font-family: \" hiragino sans gb yahei sans-serif start normal rgb><br></span></section><section style=\"margin-left: 8px;margin-right: 8px;\"><span style=\"font-size:
