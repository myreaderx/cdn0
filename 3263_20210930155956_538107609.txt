style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>显然，这是不可持续的。假如上游价格持续上涨，中游亏损压力加大，价格最终是否传递到终端，引发消费大通胀？价格传递是否顺畅取决于终端的购买力，在一般性市场，家庭购买力正常，大宗价格大涨会传递到终端引发大通胀。中国家庭的购买力无法完全承接上游价格传递，但中游制造不可能长期承担亏损，那么电力是否涨价？市场会怎么样演化？</span></p><p style=\"text-align: yahei>假设上游的价格居高不下，终端消费市场可能向局部通胀，或者“微通胀+低产能+劣商品”的组合方向演变。微通胀指的是终端市场也承接了一些价格传递（如烟酒价格上涨），低产能是上游价格太高打击了产能供给（如限电限产），劣商品是指中游将资源安排到更低品质的商品上，以满足相对疲弱的消费市场；低产能和劣商品形成一定的价格对冲，终端市场可能出现微通胀。当然，市场存在不确定性，以上仅为推测，不作为投资决策依据。</span></p><p yahei>所以，改善能源结构、保障能源供应，关键是提高能源技术与效率。能源并不存在新旧之分，不宜人为去选择或限制。煤炭、油气、光伏、风能、核能自由赛跑，市场自然会选择最有效率的能源。煤电、光伏电、风电、核电，哪个便宜又安全，哪个就是最好。假如电池储存技术获得突破，光伏电和风电则优势巨大；假如核电安全技术获得突破，将是一个重要的市场选项。也希望这次煤电供应紧张能够加速中国各类能源领域的市场化改革。</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>但是，欧洲和美国正在这个问题上犯错误，气候问题在欧美世界被政治化。</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>在欧洲，执政者长期排斥煤炭、石油，畏惧核能核电，青睐天然气，导致天然气管道受制于俄罗斯，欧洲能源价格上涨。</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>在美国，石油、天然气是共和党的支持势力，共和党反对气候政治化，特朗普政府退出巴黎气候协议。但是，对于民主党来说，气候就是政治选票，可以博取进步派背后的环保主义者及左派知识分子的支持，也可以讨好建制派背后的硅谷新能源和科技巨头。拜登上台第一件事就是让美国重返巴黎气候协议，同时打击油气势力，将页岩气的补贴转给风电和太阳能。</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>最近，拜登政府的能源部 (DOE) 发布了《太阳能未来研究》，计划到2035年，太阳能电力占据全美40%，当前这一比例仅为3%；风电和太阳能电力合计占全美75%。实际上，美国大力发展光伏发电，促使电价持续上涨。因为光伏发电的不稳定性和电力储存的高成本限制了电力供应。若要完成以上目标，按当前的储存技术，锂、铜、钴、镍等有色金属价格将难以想象。</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>这背离或干扰了能源发展的市场路线。</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>第二个一般性逻辑是解决外部性。</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>很多人会提出疑问：只关注经济效率，忽视能源安全，污染谁来负责？发生核泄漏怎么办？如何实现“碳中和”？</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>数据显示，2020年，中国二氧化碳排放大约103亿吨，其中，煤炭、石油、天然气排放达到95亿吨。换言之，90%以上的二氧化碳都来自煤炭、石油、天然气。因此，人们很自然会想办法降低或禁止这三种能源的消费。</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>但是，这并不是一个正确的逻辑。举个例子，我们每天可能都会喝牛奶，但是每多喝一瓶牛奶，就会多排放一定量的二氧化碳。这个二氧化碳主要来自：奶牛放屁、打嗝，以及牛奶生产、运输。那么，我们是不是要减少或禁止喝牛奶？</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>污染本质上是一个外部性问题，解决外部性的正确逻辑并不是消灭那个企业、产业或资源，而是让污染付出代价。比如，汽车尾气污染空气，那么使用者需要为此支付成本；工厂排污污染河流，工厂要为此支付费用。</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>解决办法可以采用庇古的征税法，如征税污染税，也可以科斯的市场交易法，如类似于排放放交易市场。让污染者为外部性付出代价，可以被迫其提高技术减少污染，最终实现“碳中和”。这样，社会福利是最大化的。只要采取解决外部性的统一标准——污染者付出代价，煤炭、石油、天然气、风电、光伏、核能都会往更加安全、更加低碳的技术方向上前进，市场个体无时无刻在边际上做出选择。比如，外部性标准促进炼油技术和发动机技术进步，如今欧洲汽车排放的二氧化碳含量已经非常低。</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top: 20px;font-family: Calibri;\"><span style=\"font-size: 16px;font-family: 微软雅黑, \" microsoft yahei>最后，除了以上两个一般性逻辑外，实际操作也考验着执政者。实际上，高福利的低碳生活是一种奢侈品。这条繁荣与低碳之路建立在技术进步与市场选择的基础之上。</span></p><p style=\"text-align: justify;font-size: 14px;line-height: 2em;margin-bottom: 20px;margin-top:
