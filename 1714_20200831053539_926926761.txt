66);\"></span></strong>步入清华科技园启迪创业孵化器的文创基地，一张张年轻的面庞映入眼帘，他们或在进行代码编辑或在线上进行客户维护，洋溢着激情与希望。主打桌游的玩亦有道科技有限公司商务经理汪洋介绍，为控制成本，公司将仓库设在湖北襄阳，这无疑造成疫情期间发货最大的不便，公司及时调整了分销策略并与合作伙伴紧急调配在全国建立了4个临时库房，才保证了及时发货。在他看来，最难的时候已经过去了，企业的前景令他和自己的团队感到鼓舞。</p><p>　　</p><p>成立于2014年的青橙创客教育，是顺利从启迪“出孵”的企业，办公室仍在清华附近，整个办公区都彰显着创始团队的工业设计背景和特色理念。</p><p><br></p><p>目前，青橙仅在北京地区建成投入使用的创新中心就有50多个，课程、活动与比赛覆盖全国700多所学校。创始人李寅也完成了从创客到创客教育工作者的完美蜕变。</p><p><br></p><p>如今在清华大学设计学专业博士在读的李寅，拥有交叉学科教育背景，他试图将技术和艺术结合在一起，借鉴STEM理念，带来跨学科的科技创新教育，培养孩子们的工程思维与设计思维。“上半年，由于学生在线学习，直接的业务量减少迫使公司进入休整期，却有了足够时间去沉淀、思考。</p><p><br></p><p>然而，休整并不意味着停滞。”李寅介绍，公司仍在持续招聘，在保有核心专家团队的基础上，调整研发人员和销售人员比例。</p><p>　　</p><p style=\"white-space: normal;max-width: 100%;min-height: 1em;letter-spacing: 0.544px;background-color: rgb(255, 255, 255);text-align: center;box-sizing: border-box !important;overflow-wrap: break-word !important;\"><strong style=\"max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\"><span style=\"max-width: 100%;font-size: 28px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">2</span></strong></p><hr style=\"white-space: normal;max-width: 100%;letter-spacing: 0.544px;text-align: center;color: rgb(62, 62, 62);font-size: 16px;background-color: rgb(255, 255, 255);box-sizing: border-box !important;overflow-wrap: break-word !important;\"><section style=\"white-space: normal;max-width: 100%;letter-spacing: 0.544px;text-align: center;background-color: rgb(255, 255, 255);box-sizing: border-box !important;overflow-wrap: break-word !important;\"><strong><span style=\"font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif justify rgb>瞄准“需求城墙口”猛攻</span></strong></section><p><br></p><p><strong><span style=\"color: rgb(171, 25, 66);\">创新是创业不竭的动力，而审时度势敏锐地瞄准社会需求、捕捉并有效地实施更是企业不断生存壮大的法则。</span></strong></p><p><br></p><p><strong><span style=\"color: rgb(171, 25, 66);\"></span></strong>在一些创业者看来，疫情的影响只是暂时的，而社会发展中涌动出来的各种需求，将是长期的，而且是迭代上升的，因此，抓住了需求，就是抓住了发展机遇。</p><p>　　</p><p>在宋军波眼中，在线教育是一个巨大的社会需求，也是推动教育变革的力量，公司5年前成立时，就是看到了这个需求的成长，而机会就是留给有准备的人的，这次疫情导致线上教育爆炸式增长，有准备的宋军波和他的企业正好接住了这个巨大的风口。</p><p><img align=\"center\" border=\"0\" data-ratio=\"0.6672413793103448\" data-type=\"jpeg\" data-w=\"580\" height=\"387\" style=\"border-width: 0px; border-style: solid; border-color: rgb(0, 0, 0); vertical-align: middle; margin: 10px auto; display: block; max-width: 600px\" title=\"\" width=\"580\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_jpg/O9F3NTo58yo5Tw2jxU9F8IqicMNteAcMwmmCKQicTbbd5UD5iak4iccY9qK7HDEntrEhDPkyPP5bzAxueKEkY6pV3w/640?wx_fmt=jpeg\"></p><p><span style=\"font-size: 14px;color: rgb(123, 127, 131);\">图为清华科技园启迪孵化器办公地 清华科技园供图</span></p><p>　　</p><p>“教育科技是全球范围内的竞争，ClassIn在这一领域完成产品‘出海’，将提升我们的全球竞争力。”宋军波介绍，翼鸥自成立之初就确定了以底层工具切入，提升线上教育教学效果的方向，而扎实的技术工具和超前的需求解决方案，让企业的产品在同类产品中具有强大的竞争力，打造行业中的领头羊产品。</p><p>　　</p><p>为青少年插上科技创新的翅膀，帮助更多孩子树立科学梦想、坚持科学梦想、实现科学梦想是青橙创客所瞄准的方向，在他们看来，这个教育需求是强劲的。</p><p><br></p><p>青橙创客鼓励孩子积极参加科技创新赛事，把在线课程与线下培训营结合起来，为孩子们开启科技创新的崭新世界。负责人李寅说，“参赛会反向促进孩子的其他文化课学习，因为研究某个领域不仅只需掌握本学科相关基础知识，也需涉猎其他交叉学科的知识。如果参加国际大赛，英文也一定要提升。因此，青橙主张把各个学科整合起来，把课堂内外结合起来。”尽管今年上半年公司业务受到疫情影响很大，但是创客的梦想不变，他们的眼睛里看到的是希望。</p><p>　　</p><p>同样的，玩亦有道创始人周歆也表示，寓教于游，是另一种教育方式。“玩亦有道的目标就是讲好中国故事，它一方面可以满足孩子玩耍的诉求，一方面可以将中华民族传统文化的精神、人文精神还有教育理念，知识点用潜移默化的方式传递给孩子。”</p><p>　　</p><p>得益于清华科技园启迪孵化器的孵化政策，玩亦有道避免了自己租办公室自己管理的琐碎，集中力量做产品研发和经营。此外，启迪孵化器还帮助对接资源和客户，让企业踏踏实实专注发展。</p><p><br></p><p>启迪孵化器总经理沈全洪说：“我们主要就是要做好两件事，一是管楼，把楼运营好，因为所有企业都离不开物理空间。第二就是服务企业，孵化器团队主要任务就是服务企业。”启迪组建了大量企业孵化器，投资新兴产业公司，进行企业投资孵化，将来会进行统一的顶层配置，快速组建产业集团。</p><p>　　</p><p !important;overflow-wrap: break-word !important;\">3</span></strong></p><hr style=\"white-space: normal;max-width: 100%;letter-spacing: 0.544px;text-align: center;color: rgb(62, 62, 62);font-size: rgb>带着“烦恼”去捕捉更大的风口</span></strong></section><p><br></p><p>今年以来，作为受疫情影响较大的群体，中小微企业的创业者面临的困境前所未有。<span style=\"font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>随着中小企业复工复产，企业信心逐步恢复，市场预期稳步向好。</span><strong style=\"font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif><span style=\"color: rgb(171, 25, 66);\">尽管如此，中小微企业仍存在抗风险能力差、资金缺乏等问题，疫后恢复和发展面临诸多困难和挑战。</span></strong></p><p><strong><span style=\"color: rgb(171, 25, 66);\"></span></strong></p><p>　　</p><p>今年2月，北京市出台了促进中小微企业发展的16条措施，从减轻中小微企业负担、加大金融支持力度等方面减轻疫情对中小微企业生产经营影响，帮助企业共渡难关。</p><p><span style=\"font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif><br></span></p><p><span style=\"font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>据悉，2018年，公司营业额为1000万，2019年攀升至2000万，而今年的目标则为3000万，现已完成1900万。</span><span style=\"font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>这个暑期，公司迎来新的爆发点，从国外引进到自主研发，从古诗词背诵到经典名著导读，周歆说：</span><span style=\"font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>“探索的路还很长。</span><span style=\"font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>”</span></p><p>　　</p><p>“ClassIn是一个集多项技术于一体的大型软件工程，在包括云计算、云通信、云存储，云通信等多项底层技术领域完成了技术突破。”宋军波也坦言，在底层技术领域，包括文字处理、浏览器等底层技术及应用仍以外来软件技术为主。“技术领域仍有较多难题需要攻关，尽管部分技术开源，但标准并不在中国企业手中，这是所有科技公司面临的问题。2020年底，翼鸥教育计划将整体人员控制在700人左右，2年内扩大到2000人的规模。”</p><p>　　</p><p>李寅认为，对青橙创客而言，现在正处于痛苦的探索期，青橙基于学科融合，并动手实践，赛道比较窄。但努力之余，却不乏欣慰，“团队这5年，大家虽然干得很辛苦，但大部分时间还是比较有成就感的。”李寅说。</p><p>　　</p><p><strong><span
