rgb(0, 0, 0);\">有人所说：“心晴的时候，雨也是晴；心雨的时候，晴也是雨。”心态好了，烦恼就少了；心态不好，处处是苦恼。</span></span></h2><h2> </h2><section style=\"white-space: normal;\"><section><section><section data-brushtype=\"text\" style=\"text-align: center;\"><span style=\"font-size: 18px;\"><strong><span style=\"color: rgb(0, 0, 0);\">1</span></strong></span></section></section></section></section><section style=\"white-space: normal;\"><p><br style=\"max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\"></p></section><section style=\"white-space: normal;\"><section data-autoskip=\"1\"><p><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　</span><span style=\"font-size: 18px;color: rgb(28, 120, 146);\"><strong>遇事时，计较不如释怀。</strong></span></p><p><br style=\"max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\"></p><p><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　生活中，难免会遇到不顺心的事。有的人事事爱计较，凡事挂心上。殊不知，这样只会让人身心俱疲、烦恼丛生。 </span></p><p><br></p><p><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　古话说：“世上无难事，庸人自扰之。”再小的事，你若计较，也会耿耿于怀；再大的事，你若淡然，也就不过尔尔。 </span></p><p><br></p><p><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　听过一个古希腊神话故事，有一位力大无穷的英雄叫海格力斯，有天他在山路上行走时，发现路中间有个袋子似的东西很碍脚，他便踢了那东西一脚。谁知那东西不但没被踢开，反而膨胀起来。海格力斯有点生气，又狠狠踩了一脚。结果那东西没被踩破，又膨胀了许多。海格力斯恼羞成怒，抄起一条粗木棒狠砸起来，那东西却加倍膨胀，最后把路堵死了。</span></p><p><br></p><p><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　这时候，山里走出来一位老人对海格力斯说：“朋友，快别动它，忽略它，离开它远去吧！它叫仇恨袋，你不犯它，它便小如当初；你的心里老记着它、侵犯它，它就会膨胀起来，挡住你前进的路，与你敌对到底！”</span><br></p><p><br></p><p><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　事情的大小，有时候不在于事情本身，而在于我们的心态。心大一点，事儿就小了；心宽一点，烦恼就少了。不斤斤计较，懂得释然，保持一颗宽容平和之心，也是遇事时聪明的处理办法。</span></p><p><br></p><section style=\"white-space: normal;\"><section><section><section data-brushtype=\"text\" style=\"text-align: center;\"><strong><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">2</span></strong></section></section></section></section><p style=\"white-space: normal;\"><br style=\"max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\"></p><p style=\"white-space: normal;\"><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　</span><strong><span style=\"font-size: 18px;color: rgb(28, 120, 146);\">失意时，生气不如争气。</span></strong></p><p style=\"white-space: normal;\"><br style=\"max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\"></p><p style=\"white-space: normal;\"><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　生活中遇到意难平的事情，与其闷闷不乐，不如着手去处理；与其火冒三丈，不如及时补救。 </span></p><p style=\"white-space: normal;\"><br></p><p style=\"white-space: normal;\"><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　战国时期著名的纵横家苏秦，早年游说六国失败后十分落魄，只好心灰意冷地回家。谁知回到家中，家人都对他冷眼相待。苏秦没有发脾气，也没有怨怼，而是反省自己的过失。他决心闭门读书、潜心学习，于是有了锥刺股的故事。后来，苏秦终于学有所成，成功游说各国合纵。 </span></p><p style=\"white-space: normal;\"><br></p><p style=\"white-space: normal;\"><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　听过这样一句话，“我几乎从来不生气，因为我认为没必要，有问题就去解决，不要让别人的错误影响自己。”</span></p><p style=\"white-space: normal;\"><br></p><p style=\"white-space: normal;\"><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　的确，发脾气是一种无用的消耗，解决不了任何问题。一个人真正成熟的表现是，生活不如意时收敛自己的脾气，不在情绪上论高下，而在能力上争输赢。</span></p><p><br></p><section style=\"white-space: normal;\"><section><section><section data-brushtype=\"text\" style=\"text-align: center;\"><span style=\"color: rgb(0, 0, 0);font-size: 18px;\"><strong>3</strong></span></section></section></section></section><p style=\"white-space: normal;\"><br style=\"max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\"></p><p style=\"white-space: normal;\"><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　</span><strong><span style=\"font-size: 18px;color: rgb(28, 120, 146);\">低谷时，抱怨不如改变。</span></strong></p><p style=\"white-space: normal;\"><br style=\"max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\"></p><p style=\"white-space: normal;\"><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　看过这样一个故事。有个叫塞尔玛的女子，陪同丈夫驻扎在一个沙漠的陆军基地里。丈夫时常有演习，塞尔玛一个人在小铁皮房子里，不仅炎热难耐，还因为语言不通，无法与人交流。她十分寂寥难受，忍不住写信向父母抱怨。</span></p><p style=\"white-space: normal;\"><br></p><p><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　 父母回了她三句话：“两个人从牢房的铁窗望出去，一个人看到了泥土，一个人看到了星星。”塞尔玛看信之后感到十分惭愧。她不再抱怨，</span><span style=\"font-size: 18px;\">决定在沙漠里找到自己的星星。</span><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">她开始学习当地语言，研究沙漠里的植物，观看沙漠日落……生活变得多姿多彩起来。后来，她把自己的生活写成了一本畅销书。 </span></p><p style=\"white-space: normal;\"><br></p><p style=\"white-space: normal;\"><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　很喜欢这样一句话：“如果生活抛给你一个柠檬，你可以把它榨成汁，然后再加点糖。”人生低谷时，与其怨天尤人，不如努力改变。若只顾着抱怨，那心里、眼里，皆是糟糕的事；若试着改变，那些苦里、难里，便能看到转机。</span></p><p style=\"white-space: normal;\"><br></p><p style=\"white-space: normal;\"><span style=\"color: rgb(0, 0, 0);font-size: 18px;\">　　生活的强者，不是一路顺风顺水，而是路难走时他们把心态调整到了最好的状态。</span><strong><span style=\"font-size: 18px;color: rgb(28, 120, 146);\">遇事时，不斤斤计较，懂得释怀，前行路上便会畅通无阻；失意时，不生气，学会争气，所经历的磨难就是成长；低谷时，不抱怨，试着改变，每一步就都是向上的路。 </span></strong></p><p><br></p><hr style=\"white-space: normal;max-width:
