","link":"http://www.geekpark.net/news/267656","description":"\n        <p>手机业务的强势，让很多人忘记了 OPPO 是一家以研发生产蓝光产品起家的公司。从 2005 年的&#160;OPPO X3 开始，之后的几年里他们陆陆续续发布了多款音频设备，也完成了早期用户积累。从这个角度来说，OPPO 做耳机类产品相比大多数手机厂商其实更有经验，算是重拾老本行。</p><p>继颈挂式主动降噪耳机 OPPO Enco Q1 和 OPPO Enco W51 等真无线耳机之后，现在 OPPO 又带来了具备主动降噪能力的真无线耳机 OPPO Enco X。它除了重点强调音质表现和降噪能力这两个大家普遍比较关注的点，诸如 ID 设计以及操控体验方面也都做了一些新的尝试。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/05/72/0572ff52b5d2919b9087c55292439295.jpg\" max-width=\"100%\" width=\"auto\" height=\"auto\">\n        </p><p><br></p><h2>精致小巧，还支持无线充电</h2><p>作为一款以「X」来命名的耳机产品，OPPO Enco X 有着很多特别的地方，首先能够体现其不同于以往&#160;OPPO 耳机的点就是充电盒。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/3b/ce/3bce9a725a94be2608a1746d5170e468.jpg\" max-width=\"100%\" width=\"auto\" height=\"auto\">\n        </p><p>\n      发布会前的官方预热中也提到过，这次 OPPO 联合了北欧高端 Hi-Fi 音响品牌丹拿（Dynaudio）一起来进行调音，所以你会在产品外包装盒以及充电盒上看到比较明显的 OPPO 和 DYNAUDIO 两个品牌 logo，而后者的加入也奠定了其本身强调音质表现的产品基调。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/b4/3a/b43a6285df33bc6ccc94aac80ebfa401.jpg\" max-width=\"100%\" width=\"auto\" height=\"auto\">\n        </p><p>细节处理上 OPPO Enco X 的充电盒也很讲究。如果你之前看到过 OPPO 初代 MP3 产品 X3 应该多多少少会有一些似曾相识的感觉，它的整体设计灵感正是来源于此，腰身一圈银色金属边框的加入，让其看上去显得精致了很多，当然也更容易握持。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/f2/5a/f25ad742196e4d884337336b4af92042.jpg\" max-width=\"100%\" width=\"auto\" height=\"auto\">\n        </p><p>唯一的一枚物理位于充电盒右侧，具体功能没啥太多可说的地方，和其它很多同类产品一样。可能看到这里大家已经发现了，OPPO Enco X 充电盒的造型设计和 AirPods 有很大区别，前者更扁平，而后者则相对「胖乎乎」一些。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/54/ce/54ce997011dedd40062727aabab22d1f.jpg\" max-width=\"100%\" width=\"auto\" height=\"auto\">\n        </p><p>它的长度为 66.3 毫米，宽度为 49 毫米，厚度为 21.9 毫米，重量为 41.5 克左右（仅充电盒）。将充电盒厚度做薄的主要好处在于方便携带，尤其是放到裤兜里时，不会出现明显鼓起的情况，平时携带起来会感觉更舒服一些。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/b5/8b/b58b6ae4960356585c8528606609fd8a.jpg\" max-width=\"100%\" width=\"auto\" height=\"auto\">\n        </p><p>至于充电盒的作用大家也都知道，除了用来收纳耳机，更重要的用处是进一步提升续航。OPPO Enco X 耳机电池容量为 44mAh，充电盒为 535mAh，支持无线充电，强降噪模式下配合充电盒使用大概能坚持 20 个小时的音乐播放时长（关闭降噪 25 个小时）。</p><p><br></p><h2>出色的音质和多种降噪模式</h2><p>归根结底，音质表现和降噪能力是评价一款真无线耳机最重要的两个维度，下面我们分别来说下 OPPO Enco X 的音质表现和降噪能力。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/44/a2/44a22bc45e4bb0503e90e12fc34e2727.jpg\" max-width=\"100%\" width=\"auto\" height=\"auto\"></p><p class=\"image-1603134423641\"></p><p>首先来看音质方面。OPPO 在这款耳机上采用了官方叫做「DBEE 3.0」的声学系统，完整叫法 Dynamic Balance Enhanced Engine 3.0 动态平衡声音引擎。其实这套声学系统并非新出现的技术名词，早年间 OPPO 就在自己的 MP3 产品上用到过，不过现在做了全面改进升级。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/a3/cf/a3cf51f1a2acc0c6de6866236343a754.jpg\" max-width=\"100%\" width=\"auto\" height=\"auto\">\n        </p><p>这套声学系统比较特别的地方在于，其将音响设备中更常见的同轴双单元设计用到了耳机产品上（此前也有耳机产品采用过类似设计，不过总体来说并不多），前置超导磁平面振膜高频单元，后置三层复合振膜大动圈中低频单元，双扬声器处于同轴。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/1c/1a/1c1a2a3950e6d1eea9dedff3c3ce5767.jpg\" max-width=\"100%\" width=\"auto\" height=\"auto\">\n        </p><p>其中超导磁平面振膜单元为超导磁微型非晶态超薄合金平膜，悬置于双排平行磁路音圈中间，由于力可以直接作用平面振膜实现同步上下振动，以此来进一步提升高频的延展性，三层复合振膜大动圈单元则用来保证中低频部分的声音表现。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/64/79/6479b1fcd9446877710026bc36e91ddf.jpg\" max-width=\"100%\" width=\"auto\" height=\"auto\">\n        </p><p>高频平面振膜单元搭配中低频动圈单元同轴双驱动，实际听下来 OPPO Enco X 给人的整体感觉是三频表现非常均衡，尤其是低频部分非常结实有力，而当你听类似《大鱼》这类歌曲时会发现，它的人声表现同样足够令人满意，解析度很高。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" SBC 和 AAC，其优点在于能够保留更多声音细节，可以带来 Hi-Res 级别的音质表现，当然这也需要设备本身支持。OPPO 手机中，目前支持 LHDC &#160;的机型包括 Find X2、Find X2 Pro、Reno 4 SE、A72 5G 等，其它机型后续会 OTA 更新支持。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/5d/ab/5dabad29c4729889701288d1967c936b.jpg\" max-width=\"100%\" width=\"auto\" height=\"auto\">\n        </p><p>降噪部分，这款耳机采用了双馈麦克风支持前馈和反馈的双馈式降噪，提供强降噪、弱降噪、降噪关以及通透模式四种降噪模式。降噪原理和其它同类产品没啥太大区别，之前我们也在很多评测文章中有过介绍，这里就不再赘述了。</p><p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" 51);\">OPPO 称，它的最大降噪深度能够达到 35db，恰好参加发布时我在飞机上也试了下其实际表现，开启强降噪模式后，对于飞机引擎声的噪音消除还是挺明显的。另外，OPPO Enco X 还采用了三麦克风通话降噪，日常通勤过程中接打电话清晰度没啥问题。</p>\n      <p style=\"text-align: center; color: #333;\"><img class=\"uploaded-img\" src=\"https://imgslim.geekpark.net/uploads/image/file/28/ba/28baf19c039ee8d8f58ff15768deb7bd.jpg\" max-width=\"100%\"
