8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">1957年，科尔夫和时年35岁的日本学者阿久津哲三</span><span style=\"font-size: 15px;color: rgb(136, 136, 136);\">（Tetsuzo Akutsu）</span><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">合作，开发出世界首个人工心脏。阿久津在其文稿中说，研发思路始于对人体心脏的模仿，“无论如何都要让人工的看起来和真正的心脏一模一样”。他在办公室里摆满机床，揉捏粘土，浇灌石膏，切割金属。他没日没夜地工作，手工制作所有零部件。最终，成品出来了，形似真实心脏，有4个腔室，包括左心房、左心室、右心房和右心室，通过使用循环液压来产生脉动流。</span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\"> </span></p><p style=\"text-align: center;margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><img class=\"rich_pages\" data-ratio=\"0.7964071856287425\" data-s=\"300,640\" data-type=\"png\" data-w=\"501\" style=\"max-width: 600px\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_png/a1xwNbPmyopA6ksiaHdGjYrRFZEyLL2byN0lwbotLDt0w5frXrYGAgDyZs9eFiaGRDmeDy63G6kfjE6Z0tweUDyg/640?wx_fmt=png\"></p><p style=\"text-align: center;margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 12px;color: rgb(136, 136, 136);\">威廉·约翰·科尔夫（左）和阿久津哲三。/Cleveland Clinic</span><br></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\"> </span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">1957年12月12日</span><span style=\"font-size: 15px;color: rgb(136, 136, 136);\">（亦有说是1958年1月）</span><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">，在体外循环的支持下，科尔夫和阿久津将其研发的人工心脏植入实验犬体内。体外循环终止后，这颗人工心继续维持血液循环90分钟。1958年，相关实验结果在美国人工器官协会第四次会议上报告。</span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\"> </span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">此后，全球越来越多的研究团队开启人工心脏尝试。</span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\"> </span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">1969年，美国完成全球首例双心室辅助装置人体植入手术。美国医生丹顿·阿瑟·库利</span><span style=\"font-size: 15px;color: rgb(136, 136, 136);\">（Denton Arthur Cooley）</span><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">为一名47岁男性，植入以其命名的“李奥塔-库利全人工心脏”</span><span style=\"font-size: 15px;color: rgb(136, 136, 136);\">（Liotta-Cooley人工心脏）</span><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">，作为心脏移植前过渡治疗。这是一颗气动的“人工心”，患者在外部动力协助下，靠它活了64小时，然后接受心脏移植。心脏移植术后32小时，患者死于假单胞菌性肺炎。</span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\"> </span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">第二例人工心脏人体植入术发生于10多年后。1981年，主刀医生仍是库利，装置是升级版、有两个气动的双心室泵</span><span style=\"font-size: 15px;color: rgb(136, 136, 136);\">（Akutsu-Ⅲ人工心脏）</span><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">。植入人体55小时后，患者接受了心脏移植。术后10天，患者死于感染、肾衰竭及肺部并发症。</span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\"> </span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">库利用过的这两种人工心脏再未被用于临床。</span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\"> </span></p><p style=\"text-align: center;margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><img class=\"rich_pages\" data-ratio=\"0.7182044887780549\" data-s=\"300,640\" data-type=\"png\" data-w=\"401\" style=\"max-width: 600px\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_png/a1xwNbPmyopA6ksiaHdGjYrRFZEyLL2byfuOwiceiaOMLNApzR7cf4bx5VWX25wicicZlbnQtksYw12bkl8frUbaF1g/640?wx_fmt=png\"></p><p style=\"text-align: center;margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 12px;color: rgb(136, 136, 136);\">图片来源于网络</span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\"> </span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">1970年，美国犹他大学推出Jarvik-7全人工心脏，依然是气动双心室泵。1982年12月2日，该装置被首次置入1名61岁心衰患者体内。术后112天，患者死亡。1985-1991年，约有170名患者使用Jarvik-7，作为心脏移植前的过渡治疗。期间，患者的败血症及多器官功能衰竭发生率较高。1991年，美国食品和药品管理局</span><span style=\"font-size: 15px;color: rgb(136, 136, 136);\">（FDA）</span><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">以不符合FDA管理条款为由，停止该装置的临床应用。</span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\"> </span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">上世纪90年代后，Jarvik-7得以革新、改名，于2001年在美国、加拿大及法国成功应用于临床。2004年，美国FDA批准其作为移植前辅助治疗措施。2012年3月，美国FDA以“人道主义使用设备</span><span style=\"font-size: 15px;color: rgb(136, 136, 136);\">（HUD）</span><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">”，批准其作为终点治疗。HUD是针对特殊医疗器械设立的，特指那些旨在服务少数特殊病患，在美国每年影响人数少于8000人，带来益处胜过使用可能造成的伤害或疾病风险。</span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\"> </span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">2000年，全球首颗全内置全人工心脏问世，组成部分包括电动泵系统</span><span style=\"font-size: 15px;color: rgb(136, 136, 136);\">（含2个人工心室和4个瓣膜）</span><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">、控制系统、电池及经皮能量传输系统</span><span style=\"font-size: 15px;color: rgb(136, 136, 136);\">（TET）</span><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">线圈。置于体内的电动泵通过TET</span><span style=\"font-size: 15px;color: rgb(0, 0, 0);\">及射频交流系统，进行充电和信号传递。因为少了穿透胸壁的管线，患者感染风险大大降低。但这一设备体积大、重900g，只适用于胸腔容积较大的患者。开发20多年来，它迟迟未得到美国FDA批准。</span></p><p style=\"margin-left: 8px;margin-right: 8px;line-height: 1.75em;\"><span style=\"font-size: 15px;color: rgb(0, 0, 0);\"> </span></p><section style=\"text-align: center;margin-left: 8px;margin-right: 8px;\"><img class=\"rich_pages\" style=\"line-height: 1.75em;text-align: center;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 12px;color: rgb(136, 136, 136);\">全球首颗全内置全人工心脏体积大，重900g。/AFP</span><br></section><p
