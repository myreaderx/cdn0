<p><b>6.3.2 Confidence Sets</b></p>\n   <p>A <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%201%20%E2%88%92%20%5Calpha\" alt=\" 1 − \\alpha\" /> <b>confidence interval</b> for a <p><img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Cmathbb%7BP%7D%28%5Ctheta%20%5Cin%20C_n%29%20%5Cgeq%201%20-%20%5Calpha\" alt=\" \\mathbb{P}(\\theta \\in C_n) \\geq 1 - \\alpha\" This is correct but useless since we rarely repeat the same experiment <a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/over%20and%20over\">over and over</a>. A better interpretation is this:</p> \n   <blockquote> \n    <p>On day 1, you collect data and construct a 95 percent confidence interval for a parameter <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta_1\" alt=\" \\theta_1\" />. On day 2, you collect new data and construct a 95 percent confidence interval for an unrelated parameter <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta_2\" alt=\" \\theta_2\" />. On day 3, you collect new data and construct a 95 percent confidence interval for an unrelated parameter <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta_3\" alt=\" \\theta_3\" />. You continue this way constructing confidence intervals for a sequence of unrelated parameters <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta_1%2C%20%5Ctheta_2%2C\" alt=\" \\theta_1, \\theta_2,\" />... Then 95 percent of your intervals will trap the true parameter <a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/value\">value</a>. If you form a confidence interval this way every day for the rest of your life, 95 percent of your intervals will contain the true parameter. This is true even though you are estimating a different quantity (a different poll question) every day. ■</p>\n   <br /> \n   <p><b>6.14 Example.</b> The fact that a confidence interval is not a probability statement about <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta\" alt=\" \\theta\" /> is confusing. Let <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta\" alt=\" \\theta\" /> be a fixed, known real number and let <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20X_1%2C%20X_2\" alt=\" X_1, X_2\" /> be independent random variables such that <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Cmathbb%7BP%7D%28X_i%20%3D%201%29%20%3D%20%5Cmathbb%7BP%7D%28X_i%20%3D%20-1%29%20%3D%201%2F2\" alt=\" \\mathbb{P}(X_i = 1) = \\mathbb{P}(X_i = -1) = 1/2\" />. Now define <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20Y_i%20%3D%20%5Ctheta%20%2B%20X_i\" alt=\" Y_i = \\theta + X_i\" /> and suppose that you only observe <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20Y_1\" alt=\" Y_1\" /> and <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20Y_2\" alt=\" Y_2\" />. Define the following “confidence interval” which actually only contains one point:</p>\n   <br /> \n   <p><img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20C%20%3D%20%5Cleft%20%5C%7B%20%5Cbegin%7Barray%7D%7Bl%7D%20%20%5C%7B%20Y_1%20-%201%20%5C%7D%20~%20if%20~%20Y_1%20%3D%20Y_2%20%5C%5C%20%5C%7B%20%28Y_1%20%2B%20Y_2%29%20%2F%202%20%5C%7D%20~%20if%20~%20Y_1%20%5Cneq%20Y_2%20%5Cend%7Barray%7D%20%5Cright.\" alt=\" C = \\left \\{ \\begin{array}{l}  \\{ Y_1 - 1 \\} ~ if ~ Y_1 = Y_2 \\\\ \\{ (Y_1 + Y_2) / 2 \\} ~ if ~ Y_1 \\neq Y_2 \\end{array} \\right.\" /></p>\n   <br /> \n   <p>You can check that, no matter what <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta\" alt=\" \\theta\" /> is, we have <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Cmathbb%7BP%7D%20%5Ctheta%20%28%5Ctheta%20%5Cin%20C%29%20%3D%203%2F4\" alt=\" \\mathbb{P} \\theta (\\theta \\in C) = 3/4\" /> so this is a 75 percent confidence interval. Suppose we now do the experiment and we get <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20Y_1%20%3D%2015\" alt=\" Y_1 = 15\" /> and <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20Y_2%20%3D%2017\" alt=\" Y_2 = 17\" />. Then our 75 percent confidence interval is {16}. However, we are certain that <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta%20%3D%2016\" alt=\" \\theta = 16\" />. If you wanted to make a probability statement about <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta\" alt=\" \\theta\" /> you would probably say that <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Cmathbb%7BP%7D%28%5Ctheta%20%5Cin%20C%20%7C%20Y_1%2C%20Y_2%29%20%3D%201\" alt=\" \\mathbb{P}(\\theta \\in C | Y_1, Y_2) = 1\" />. There is nothing wrong with saying that {16} is a 75 percent confidence interval. But is it not a probability statement about <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta\" alt=\" \\theta\" />. ■</p>\n   <br /> \n   <p>In Chapter 11 we will discuss Bayesian methods in which we treat <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta\" alt=\" \\theta\" /> as if it were a random variable and we do make probability statements about <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta\" alt=\" \\theta\" />. In particular, we will make statements like “the probability that <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta\" alt=\" \\theta\" /> is in <img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20C_n\" alt=\" C_n\" />, given the data, is 95 percent.” However, these Bayesian intervals refer to degree-of-belief probabilities. />を越えるものも含めて考えるためである。</p>\n   <br /> \n   <p><b>（9.3節 信頼<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%B6%E8%B4%D6\">区間</a>の解釈）</b></p>\n   <br /> \n   <p>9.1節で述べたように信頼域については</p>\n   <br /> \n   <p><img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20P_%7B%5Ctheta%7D%20%28%5Ctheta%20%5Cin%20S%28X%29%29%20%3D%201%20-%20%5Calpha\" alt=\" P_{\\theta} (\\theta \\in S(X)) = 1 - \\alpha\" />　(9.19)</p>\n   <br /> \n   <p>を「確率」とよばずに「信頼係数」とよんでいる。それは<img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20P_%7B%5Ctheta%7D%20%28%5Ctheta%20%5Cin%20S%28X%29%29\" alt=\" />に含まれる確率を議論することはできない。すなわち<img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta\" alt=\" \\theta\" />は未知ではあるが固定された値であるから、特定の<img <p>……通常の信頼域については事後的な確率は未知である。信頼域が母数を含む事後確率の期待値が信頼係数であるから、事後確率についてはっきりした情報がなければ、いわば事後確率の推定値として信頼係数を用いることも合理的であると考えられる。また、同じ信頼係数の信頼域を異なるデータに対して繰り返し構成する場合を考えれば、信頼係数は信頼域が母数を含む比率となるから、1回ごとの繰り返しがお互いに区別できないという前提のもとでは、信頼係数を事後的な確率と考えることも正当化されよう。（中略）</p>\n   <br /> \n   <p>信頼係数の解釈については<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%A5%D9%A5%A4%A5%BA\">ベイズ</a><a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%C5%FD%B7%D7%B3%D8\">統計学</a>の立場からの解釈も重要である。実は<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%A5%D9%A5%A4%A5%BA\">ベイズ</a><a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%C5%FD%B7%D7%B3%D8\">統計学</a>の立場にたてば、母数が信頼域に含まれる事後的な確率を明示的に求めることができ、信頼係数を考える必要はない。これは、<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%A5%D9%A5%A4%A5%BA\">ベイズ</a><a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%C5%FD%B7%D7%B3%D8\">統計学</a>においては、<img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta\" alt=\" \\theta\" />は固定された値ではなく確率変数と解釈するからである。<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%A5%D9%A5%A4%A5%BA\">ベイズ</a><a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%C5%FD%B7%D7%B3%D8\">統計学</a>の考え方については14章で論じている。</p>\n   <br /> \n   <p><b>（14.2 事前分布と事後分布）</b></p>\n   <br /> \n   <p>信頼<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%B6%E8%B4%D6\">区間</a>について注意すべき点は、<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%A5%D9%A5%A4%A5%BA\">ベイズ</a>信頼<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%B6%E8%B4%D6\">区間</a>においては母数<img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20p\" alt=\" p\" />は確率変数であるから、<img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20X%20%3D%20x\" alt=\" X = x\" />を観測した条件のもとで信頼<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%B6%E8%B4%D6\">区間</a>が母数<img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20p\" alt=\" p\" />を含む確率を文字通りに解釈できるという点である。これに対して、古典的な信頼<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%B6%E8%B4%D6\">区間</a>においては、すでに9.3節で述べたように、信頼<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%B6%E8%B4%D6\">区間</a>の信頼係数を事後的な確率と解釈することはできない。このように<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%A5%D9%A5%A4%A5%BA\">ベイズ</a>法においては概念的に矛盾のない信頼<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%B6%E8%B4%D6\">区間</a>を構成することができる。</p>\n   <p>（太字<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%B8%B6%CA%B8%A5%DE%A5%DE\">原文ママ</a>）</p> \n  </blockquote> \n  <p><br /> 重要なポイントは9.3節で、「<img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20P_%7B%5Ctheta%7D%20%28%5Ctheta%20%5Cin%20S%28X%29%29\" alt=\" P_{\\theta} (\\theta \\in S(X))\" />（例えば「95%」）を確率と解釈することについて問題がある」「これはいわば<img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20X\" alt=\" X\" />を観測する前の事前の確率であり、特定の<img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20X%20%3D%20x\" alt=\" X = x\" />が実現したあとに<img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20%5Ctheta\" alt=\" \\theta\" />が<img src=\"https://chart.apis.google.com/chart?cht=tx&amp;chl=%20S%28x%29\" alt=\" S(x)\" />に含まれる確率を議論することはできない」という説明が全てだと思います。そして「同じ信頼係数の信頼域を異なるデータに対して繰り返し構成する場合を考えれば、信頼係数は信頼域が母数を含む比率となる」ことから、「95%」が「幅」「割合」たり得ることも読み取れます。ただし、同じ箇所で「信頼域が母数を含む事後確率の期待値が信頼係数であるから、事後確率についてはっきりした情報がなければ、いわば事後確率の推定値として信頼係数を用いることも合理的であると考えられる」と述べているように、「95%」を「事後確率の推定値」即ち「確率」として扱うことを全否定しているわけでもないことが分かります。</p>\n  <br /> \n  <p>とは言え、後の14.2節で改めて<a class=\"keyword\" href=\"http://d.hatena.ne.jp/keyword/%A5%D9%A5%A4%A5%BA\">ベイズ</a>信頼<a class=\"keyword\"
