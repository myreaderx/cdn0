Abra um terminal (No Unity, use o Dash ou pressionando as teclas CTRL+ALT+T);<br />Passo 2. Confira se o seu sistema é de 32 bits ou 64 bits, para isso, use o seguinte comando no terminal:</p>\n<pre>\n<code>uname -m</code>\n</pre>\n<p>Passo 3. Agora instale o programa com o comando:</p>\n<pre>\n<code>sudo dpkg -i minitube.deb</code>\n</pre>\n<p>Passo 6. Caso seja necessário, instale as dependências do programa com o comando:</p>\n<pre>\n<code>sudo apt-get install -f</code>\n</pre>\n<p>Passo 7.
