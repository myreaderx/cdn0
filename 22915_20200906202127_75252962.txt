This new feature allows you to search through all of the commands available to you in Windows Terminal, similar to the one found in Visual Studio Code. You can invoke the command palette by typing <code>Ctrl+Shift+P</code>. If you&#8217;d like to change this key binding, you can add the <code>commandPalette</code> command to the <code>keybindings</code> array in your settings.json.</p>\n<pre><code>{ \"command\": \"commandPalette\", \"keys\": \"ctrl+shift+p\" }\n</code></pre>\n<p>The command palette has two modes: action mode and command line mode. Action mode is the mode you will enter by default and will list all of your Windows Terminal commands. Command line mode can be entered by typing <code>&#62;</code> and you can then enter any <code>wt</code> command, which will be invoked on the current window.</p>\n<p>You can also customize actions you&#8217;d like to add to the command palette by adding commands to your settings.json file. Your key bindings should automatically populate the command palette.
