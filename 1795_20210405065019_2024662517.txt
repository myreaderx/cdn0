style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">你们懂得撒了谎必须道歉这个非常简单的道理吗？</span></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">罗莱娜·博蒙文章：</span><span style=\"letter-spacing: 0.544px;text-align: center;\"></span></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb center border-box break-word><strong style=\"max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\"><span style=\"max-width: 100%;font-size: 18px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">（看中国）一些西方媒体描述的新疆和我所见所闻迥然不同</span></strong></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb center border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">作者：罗莱娜·博蒙（Laurène Beaumond），独立撰稿人，法语部海外评论员</span></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb center border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">翻译：朱丹</span></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">这个世界是不是丧失了理智？“集中营”、“强制妇女绝育”、“强制劳动”、“文化灭绝”、“种族清洗”……，运用这些人类历史上最黑暗的字眼，西方发起了一场针对中国的远程诉讼，尽管一无确凿证据、二无可靠证词，尽管发起诉讼的人从未踏上过这片土地——中国的新疆维吾尔自治区，有些人甚至在今年年初之前都没有听说过这个地方。他们为什么发起这场闹剧？瑞士“良好棉花发展协会”（BCI）声称新疆存在“强迫劳动”，欧美知名制衣品牌配合选边站队，宣布停止使用新疆棉花，这一事件成为引起一场大风波的最后一滴水。</span></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">西方对新疆的攻击着实令人咋舌：中国，一个从不惹事的国家，无端地被置于猛烈的舆论炮火之下，各种针对中国的指责充满了荒谬的想象。这些事件严重损害了欧盟和中国的关系，尽管双方曾在去年年末达成了一项历史性的协议——中欧投资协定谈判如期完成。如今，新冠疫情阴霾尚未消散，中欧理应携手并进，而不是分道扬镳。</span></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">我是一名曾在中国生活了7年的法国人，我正好有家人居住在新疆维吾尔自治区首府乌鲁木齐。2014年至2019年间，我多次到访新疆，一些西方媒体描述的新疆和我所认识的新疆迥然不同。我在这个地区的旅行都是自发自愿的，既不是出于职业目的，更不是由任何组织“策划”的，我也没有参加任何一个旅行社。在踏上新疆这片土地之前，我对它的了解仅限于阅读历史小说、去博物馆欣赏文物宝藏。保存在巴黎吉梅特博物馆（Guimet Museum）的6世纪库车佛教壁画让我印象深刻。我知道新疆是一个文化交融的地区，在那里有许多民族混居在一起：当然有维吾尔族，也有蒙古族、俄罗斯族、哈萨克族……。这块广阔的土地还是古今丝绸之路的核心地带，各民族在这里融合聚居，各族文化在这里交流汇合。</span></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">新疆带给我的首先是惊喜。我停留最久的地方是乌鲁木齐，也去过喀什和阿克苏。喀纳斯的美丽让我难以忘怀，那里的绿色山谷让人想起瑞士的阿尔卑斯山。新疆带给我的感觉，就和我去内蒙古自治区或吉林延边朝鲜族自治州旅行时的感觉一样：和谐包容、各民族相互尊重，最重要的是那里的居民对大自然的尊重和依恋。文化灭绝？怎么可能。在新疆，所有商店标牌都有两种语言：汉语和维吾尔语。所有的行政文件也都有两种语言的版本。2016年，因为健康出了点问题，我曾在乌鲁木齐住院一周。在这期间，几个维吾尔族医生负责我的诊疗工作，这家医院就在乌鲁木齐市最大的清真寺旁边。每天早上，我都在新疆穆斯林的祷告声中醒来，医院的食堂提供百分之百的纯清真食品。后来，当我看到法语维基百科页面介绍新疆的文字，我陷入了震惊：“2010年，数十万维吾尔族穆斯林……在中国被关进‘集中营’，被迫接受再教育……。”如果你没去过新疆，这段言之凿凿文字看上去是那么让人信服，作者让人觉得他们的信息来源是那么可靠。</span></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">在新疆，我曾和一个哈萨克族警察共进晚餐，他是当地特种部队的成员（幸运的是他会说英语）。我问他，少数民族在加入当地警察或者部队时是否会遇到障碍。“没有！”他回答说。“关于新疆的谎言实在太多了，造谣的人甚至根本意识不到自己在说谎……。”这段经历发生在2015年，我认为新疆当地各民族共居、融合的趋势从未改变。在清真寺、大巴扎，保存完好的传统伊斯兰手工艺品随处可见。</span></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">我并不是说，凭借不到十次到访新疆的经历，我就了解了那里的一切。但我在那里所看到的、所经历的，与西方媒体报道的完全不符。在乌鲁木齐，我有汉族朋友，也有维吾尔族朋友。汉族人说他们“热爱乌鲁木齐的生活，不会无缘无故离开”，维吾尔族人亦如是说。在乌鲁木齐到北京的航班上，我遇到一对法国游客，他们告诉我“新疆是他们最喜欢的中国省份”。在我到访新疆的时候，无论是在城市还是农村，我能感觉到这里的人们发自内心的快乐。各族人民经营着他们的小生意，穆斯林群众有宗教信仰自由，按照他们自己的喜好穿衣打扮。对我这个异乡人，他们充满热情地欢迎。</span></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">新疆棉花被扣上了“强迫劳动”的帽子，这简直太荒谬了。维吾尔族工人被强迫在田里采摘棉花？看来有些人永远忘不了美国奴隶制的罪恶，一提到棉花就想当然地对号入座。荒唐的谎言无法扼杀真相。在新疆辽阔的土地上，棉花生产早已实现高度机械化。</span></p><p style=\"margin-bottom: 20px;max-width: 100%;min-height: 1em;font-family: -apple-system, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif normal rgb border-box break-word><span style=\"max-width: 100%;font-size: 16px;box-sizing: border-box !important;overflow-wrap: break-word !important;\">我是法国人，我为此感到自豪。换位思考，我也不喜欢我的国家受到不公正的批判。当某些媒体、机构、组织胡乱评说新疆的时候，中国人难道就没有权利为此感到愤怒吗？为什么我们不能听听中国人对这件事的说法呢？中国人是当事人，他们难道连质疑谣言的权利都没有吗？就我个人而言，我迫不及待地等待疫情结束，然后回到新疆，和我的家人一起团聚欢笑。</span></p><p
