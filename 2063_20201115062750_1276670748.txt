16px;\">7年后的1722年，小国王达到法定的成年年纪，摄政王的当权生涯即将结束，国王的车队再次抵达凡尔赛，市民们欢欣鼓舞，要求燃放烟火。路易十五居住凡尔赛宫期间，宫中的附属机构、王公大臣和仆人大约有4000人。凡尔赛宫里人满为患，每个住在这里的贵族都有自己的仆人和轿夫，各个长廊都站满了人。住房又开始紧张起来，贵族大臣们想在宫里弄套住房可不容易，凡尔赛城镇区的房屋又奇货可居起来。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"> </span></p><section style=\"box-sizing: border-box;font-size: 16px;\"><section style=\"box-sizing: border-box;\" powered-by=\"xiumi.us\"><section style=\"display: flex;flex-flow: row nowrap;margin: 10px 0%;text-align: center;justify-content: center;box-sizing: border-box;\"><section style=\"display: inline-block;width: auto;vertical-align: bottom;box-shadow: rgb(0, 0, 0) 0px 0px 0px;border-style: solid;border-width: 10px;border-radius: 0px;border-color: rgba(241, 238, 190, 0.22);flex: 0 0 auto;align-self: flex-end;min-width: 10%;max-width: 100%;height: auto;box-sizing: border-box;\"><section style=\"justify-content: center;margin-right: 0%;margin-left: 0%;box-sizing: border-box;\" powered-by=\"xiumi.us\"><section style=\"display: inline-block;width: 100%;vertical-align: top;height: auto;background-color: rgba(248, 242, 149, 0.76);border-style: solid;border-width: 1px;border-radius: 0px;border-color: rgb(255, 255, 255);box-sizing: border-box;\"><section style=\"text-align: justify;color: rgb(0, 0, 0);line-height: 2;letter-spacing: 0px;padding-right: 10px;padding-left: 10px;box-sizing: border-box;\" powered-by=\"xiumi.us\"><p style=\"white-space: normal;box-sizing: border-box;\">国家经济的重负</p></section></section></section></section></section></section></section><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><br></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">路易十五执政后，重新启用了凡尔赛宫，他按照自己的意愿改造了一些结构，把靠近内阁会议厅的房间当作寝宫，因为原来那间国王寝宫太冷了，导致其经常感冒。国王早上不再有固定的起床时间，侍从不准叫醒他。他总是深夜不眠，彻夜玩乐，与情妇约会，玩牌，也开舞会，经常带着随从悄悄溜出去，清晨六时再敲响城堡的大门：“开门，我是国王！”</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><section style=\"text-align: center;margin-left: 8px;margin-right: 8px;\"><img class=\"rich_pages js_insertlocalimg\" data-ratio=\"1.2075471698113207\" data-s=\"300,640\" data-type=\"png\" data-w=\"636\" style=\"width: 341px; height: 412px; max-width: 600px\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_png/laADTC8BCK70g7SZicRxmZ76K7T9EiamSqxncyNY44c7sOt9wicicc1ialeibIv4aAJyUWCUXRAV82sBk0PaBphNEIog/640?wx_fmt=png\"></section><section style=\"text-align: center;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 14px;color: rgb(136, 136, 136);\">路易十五</span></section><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><br></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">玛丽王后为路易十五生育了一堆孩子，起初几年国王对王后忠贞不移，但花花公子性格使他很快拥有一堆情妇，诸如蓬帕杜尔夫人和妓女出身的巴莉夫人等等。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">凡尔赛宫的生活周而复始，国王晚年不再公开找情妇，而是置办了臭名昭著的鹿园，以供那些国王的年轻女子居住。法兰西的政权依旧设立在凡尔赛，1768年起，内阁的主要部门也在凡尔赛落户。各部大臣不必每日奔波于巴黎和凡尔赛之间。外省贵族们每年都削尖了脑袋要在凡尔赛宫住上两三个礼拜，以供其回家乡后炫耀。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">凡尔赛宫再次迎来喧哗热闹的场面是王子</span><span style=\"font-size: 14px;color: rgb(136, 136, 136);\">（未来的路易十六）</span><span style=\"font-size: 16px;\">的婚礼：婚礼仪式、晚宴、舞会、演出……这些活动犹如路易十四在时那样豪华气派。这场婚礼的总花销是900万里弗尔</span><span style=\"font-size: 14px;color: rgb(136, 136, 136);\">（约3600万法郎）</span><span style=\"font-size: 16px;\">。路易十五曾问他的财政大臣，对自己举办这场婚礼有什么感想？回答是：“我觉得它是……难以支付的。”如贵族们一贯的做法，这场豪华婚礼大部分靠举债，一直到20年后的法国大革命时期，依然有人手持着备忘录，要求国王先付上一点欠款。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">1774年，64岁的路易十五依然精力充沛，一连几个小时打猎都不觉得疲惫。突然有一天早上醒来，他感到腰酸背痛浑身打寒战，不久全身布满了丘疹，首席医生最后检查的结果是——天花。当时，接种牛痘以预防天花已经出现，但法国医学院十分保守，路易十五也不愿意接受这种新鲜事物，最终死于这种疾病。几个月后，继位的路易十六开始带头接种牛痘。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">历史学家对路易十五的评价十分严厉。他穷兵黩武，庞大的军费开支拖垮国内经济；对民众课以重税，榨干平民；一手缔造并维护贵族阶层特权。临终前，他曾借由大臣的口对臣民忏悔。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"> </span></p><section style=\"box-sizing: border-box;font-size: 16px;\"><section style=\"box-sizing: border-box;\" powered-by=\"xiumi.us\"><section style=\"display: flex;flex-flow: row nowrap;margin: 10px 0%;text-align: center;justify-content: center;box-sizing: border-box;\"><section style=\"display: inline-block;width: auto;vertical-align: bottom;box-shadow: rgb(0, 0, 0) 0px 0px 0px;border-style: solid;border-width: 10px;border-radius: 0px;border-color: rgba(241, 238, 190, 0.22);flex: 0 0 auto;align-self: flex-end;min-width: 10%;max-width: 100%;height: auto;box-sizing: border-box;\"><section style=\"justify-content: center;margin-right: 0%;margin-left: 0%;box-sizing: border-box;\" powered-by=\"xiumi.us\"><section style=\"display: inline-block;width: 100%;vertical-align: top;height: auto;background-color: rgba(248, 242, 149, 0.76);border-style: solid;border-width: 1px;border-radius: 0px;border-color: rgb(255, 255, 255);box-sizing: border-box;\"><section style=\"text-align: justify;color: rgb(0, 0, 0);line-height: 2;letter-spacing: 0px;padding-right: 10px;padding-left: 10px;box-sizing: border-box;\" powered-by=\"xiumi.us\"><p style=\"white-space: normal;box-sizing: border-box;\">最后的锁匠国王</p></section></section></section></section></section></section></section><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><br></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">路易十五的葬礼搞得浮皮潦草，朝廷为国王治丧，大家心里却笑开了花。他的继任者路易十六仍然选择住在凡尔赛宫，年轻的国王不如路易十四那样张扬、也不像父亲路易十五那样风流成性，是个细致入微的工科男。1787年，地理学家新绘制了一张彩色城堡图，交由国王审阅。路易十六认真研读后，用笔工整地记下地图上的错误，并写道：“在莫勒帕大街和公园之间的那些小花园并不相同——池塘没有那么大，还有一片荒地被遗忘了。”</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">他把原先路易十五的赌博间改成自己的书房，入驻凡尔赛宫后的大部分时间都用来研究机械，更确切地说是制锁。路易十六是典型的工科狂人，他对锁的研究已不能用单纯的专业来衡量。他的锁形状各异、极富创意，几乎每一把都是一件艺术品。他曾把锁制成活泼可爱的鲤鱼、松鼠或鸭子形状，扭动“松鼠”的钥匙，“松鼠”会频频点头，摇尾乞怜。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">路易十六在凡尔赛宫里专门建造了一间全法国最高级的五金作坊，这间面积近百平方米的房间内挂满了各种工具。为制造最复杂、最精美的锁，他高薪聘请了民间著名的铜匠加曼，后者可以自由出入他的寝宫。据说路易十六的开锁技巧十分高超，可以在锁芯弹回的零点几秒内拧开锁。他曾经制造了一把最得意的锁，收藏国家财务和镇压革命军等绝密文件，他认为世上没有其他人能打开它。在路易十六被囚禁后，人们找到这个箱子，由于锁极为精密而箱子又十分牢固，万般无奈下只能请来加曼开锁。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">路易十五晚期和路易十六早期，凡尔赛宫的规模达到史上巅峰，人员配置更臃肿，浪费严重。维护凡尔赛宫日常的开支占到法国岁入的1/4。例如在王妃小寝宫点的蜡烛，主人一离开就要熄灭换新的，旧的只烧掉一点点，很多王室的新衣只穿一次就被丢弃。转卖皇宫小物件已再平常不过，各种私下买卖慢慢腐蚀这个庞大的机器。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">1789年，攻占巴士底狱的那个夜晚，睡眼惺忪的路易十六问：“这是一次暴乱吗？”“不，陛下，这是一次革命。”</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">对于大厦将倾，路易十六并不是毫无察觉，但他所立的改革措施都半途而废，国家危机积重难返，他本人又过于优柔寡断。茨威格的《断头王后》一书，在“凡尔赛的最后一夜”一章中写道：惊慌失措的国王站在那里，一脸恐惧。距离人民部队攻入凡尔赛宫还有两个小时，他们还有时间作决策。一位大臣建议国王应该骑上战马带领军队去战斗；保守派则认为国王和王后应该立刻离开王宫，前往朗布依埃。但此时的路易仍然犹豫不决。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">10月5日下午，革命人群打开凡尔赛宫的大门，国王和王后在阳台出现，他们坐上备好的马车，被迫向巴黎驶去。临走时路易十六对他的作战大臣拉图尔·杜班说：“您仍然是这里的主人，要尽力把我那可怜的凡尔赛宫保留下来。”凡尔赛宫作为国王行宫的一百多年历史随着旧政权的倒台戛然而止。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">在此后的法国大革命期间，凡尔赛宫被洗劫一空，1793年，这里残存的艺术品被运送至卢浮宫保存。1833年，奥尔良王朝的路易·菲利普国王下令修复这座废墟，改建为历史博物馆。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">1870年，普鲁士军队占领凡尔赛，第二年德皇在此举行加冕典礼。同年，梯也尔政府盘踞公主的套房，在凡尔赛宫策划了镇压巴黎公社的血腥计划。1919年6月28日，法国及英美等国同德国在这里签订了宣告第一次世界大战结束的《凡尔赛和约》。</span></p><p style=\"text-align: justify;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\"><br></span></p><section style=\"text-align: center;margin-left: 8px;margin-right: 8px;\"><img class=\"rich_pages js_insertlocalimg\" data-ratio=\"0.678125\"
