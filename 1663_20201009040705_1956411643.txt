16px;\">而京东健康自诞生以来就一直处于盈利状态，其营收规模也早已突破百亿，规模处于行业领先地位。除去A轮优先股公允价值变动，京东健康2017-2020H1调整后净利润分别达到2.09亿元、2.48亿元、3.44亿元以及3.71亿元。</span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">从主营业务上来看，与京东健康最接近的就是阿里健康。二者均涉猎自营及平台医药电商、消费医疗、互联网医疗和数字医疗；阿里健康2020财年总营收达到96亿元，医药电商业务贡献了97%，而京东健康这一比例已经下降至87%左右。这也使得截至2019年，京东健康营收108亿元，净利3.44亿元，赚钱效应更加明显，从盈利角度，财务模型显著优于阿里健康。</span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">业务拆分以来，二者的较量已经延伸至物流、金融、互联网医疗等行业。根据最新的公开数据：菜鸟和京东物流的估值都接近2000亿元；蚂蚁集团估值约2000亿美元，京东数科估值约2000亿元。作为传统电商行业的头部玩家，阿里的市值也是京东的6倍，可以说阿里仍绝对领先京东。</span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">但京东健康的业务范畴和规模都超过阿里健康，这次明显前者更胜一筹。阿里健康上市6年以来，市值已经达到300亿美元，今年市值已经翻倍。2019年市场对京东健康的估值是70亿美元，借鉴阿里健康今年23倍的市销率，京东健康的估值至少达到300亿美元。可以说，京东健康的IPO带来了今年医疗健康赛道最大的黑马。</span></p><p><span style=\"font-size: 16px;\"> </span></p><p><strong><span style=\"font-size: 16px;\">京东集团的交易规模和利润都弱于阿里巴巴，前者市值只是后者的七分之一。但京东健康则在规模和盈利能力上都强于阿里健康，从竞争力上来看，医药更加依赖自营模式，所以长远来看京东健康将比阿里健康更有优势。海豚智库预计，京东健康IPO后，市值有望超过阿里健康。</span></strong></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\"><strong style=\"font-size: 16px;white-space: normal;max-width: 100%;color: rgba(0, 0, 0, 0.8);font-family: -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif rgb border-box break-word><span style=\"max-width: 100%;color: rgb(0, 122, 170);box-sizing: border-box !important;overflow-wrap: break-word !important;\"><strong style=\"max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\">►</strong></span></strong> </span><strong><span style=\"font-size: 16px;color: rgb(0, 122, 170);\">百舸争流，京东健康攻下一城的底气何在？</span></strong></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">在医药零售和互联网医疗服务协同下，京东健康背后的两大关键实力不言而喻——供应链能力和线上医疗服务能力。</span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">在供应链方面，截至2020年8月，京东大药房在全国范围内拥有11座药品专用仓库，分布国内七大区域核心城市，非药品仓库超过230个，全国范围内可实现80%订单次日达，具备全行业最快的履约服务能力。京东健康的O2O服务“京东药急送”已经覆盖超200个城市，能提供全天候30分钟送达、7*24快递服务，满足用户紧急性用药需求。这背后，亦离不开京东十年磨一剑的“京东物流”。</span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">在医疗服务方面，京东互联网医院是国内首批取得互联网医院牌照的平台型互联网医院之一，也是增速最快的互联网医疗平台之一。开设了十六大专科中心，有近百位权威专家和顶级名医。</span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">利用自身强大的供应链和技术优势，京东健康为整个医疗健康产业链上的其他参与者提供数字化的解决方案；通过AI、大数据等在内的领先技术，京东健康积极研发落地下一代技术应用，例如智能问诊助手、智能处方审核助手，智能健康设备等。京东健康将各类应用与云技术基础设施结合，为线下医院提供涵盖其各方面服务的综合解决方案。同时，京东健康也为多个地方政府提供“健康城市”解决方案，涵盖基层医疗、居民健康档案、医保支付等领域。</span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">从零售的逻辑转向服务的逻辑，京东对于京东健康这块业务真正的立意，本质还是通过数字技术的“穿透性”支撑业务的协同性，提升B端和C端服务的品质和效率。</span></p><p><span style=\"font-size: 16px;\"> </span></p><p><span style=\"color: rgb(0, 122, 170);\"><strong><span style=\"font-size: 16px;\"><strong style=\"font-size: 16px;white-space: normal;max-width: 100%;color: rgba(0, 0, 0, 0.8);font-family: -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif rgb border-box break-word><span style=\"font-size: 16px;max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\"><strong style=\"max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\"><strong style=\"font-size: 16px;white-space: normal;max-width: 100%;color: rgba(0, 0, 0, 0.8);font-family: -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif rgb border-box break-word><span style=\"max-width: 100%;color: rgb(0, 122, 170);box-sizing: border-box !important;overflow-wrap: break-word !important;\"><strong style=\"max-width: 100%;box-sizing: border-box !important;overflow-wrap: break-word !important;\">►</strong></span></strong></strong></span></strong></span></strong></span><strong><span style=\"font-size: 16px;color: rgb(0, 122, 170);\"> 互联网医疗千亿蓝海之下，成就京东的第四驾马车</span></strong></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">近年来，国家密集发布了一系列的“互联网+医疗健康”的政策，行业发展迎来政策红利期。互联网医疗市场规模、互联网医院数量和问诊量都实现跨越式增长，医药电商市场规模、用户数量增长迅速。</span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">根据易观数据，2019年中国移动医疗市场规模为1336亿元，同比增长35.6%，今年因疫情影响，市场规模有望达到2000亿，增速达到新高至46.7%。医疗这块蛋糕，作为基本民生服务，线上化的进程才刚刚开始，规模有望在接下来5年内，迅速扩张至万亿级。</span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">尤其在疫情催化下，互联网医疗平台用低成本完成了用户教育，“互联网+医疗健康”产业快速发展，备受资本市场的关注。不过，难点在于疫情过后的用户留存问题。</span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">京东健康致力于建设完整、全面的“互联网+医疗健康”产业生态，为整个“互联网+医疗健康”产业生态中的所有参与者均创造了巨大价值。</span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><strong><span style=\"font-size: 16px;\">对于用户而言，京东健康缓解了医疗资源供给不平衡的痛点，带来了优质的健康产品和服务。</span></strong></p><p><strong><span style=\"font-size: 16px;\"><br></span></strong></p><p><strong><span style=\"font-size: 16px;\">对于药企、医疗器械供应商等第三方商家来说，京东健康帮助缩减了医疗健康产业链中间环节，也可以为药品和医疗器械供货商提供反向定制，助其产品研发。</span></strong><span style=\"font-size: 16px;\"></span></p><p><span style=\"font-size: 16px;\"><br></span></p><p><span style=\"font-size: 16px;\">现在，在互联网医疗这个千亿蓝海里，巨头涌进，不过淘金却并没有想象中的容易，京东健康是屈指可数的已经实现盈利的玩家。京东在打造了三辆马车之后，京东健康势必能成为第四辆马车。</span></p><p><br></p><p style=\"margin-top: 5px;padding-top: 10px;padding-bottom:
