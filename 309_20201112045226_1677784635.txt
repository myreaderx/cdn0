form”表示。这类谱例的音符记写仍然偏向传统，但乐曲不再是方方正正、连贯成行的式样，而是被分割成许多小部分杂乱放置，视觉效果与传统记谱非常不同。这些小部分通常会被标号、排序，演奏时需得按照每个小部分末端的指定序号进行下一个片段的演奏，就像环环相扣的解谜游戏，需要演奏者“及时跳转、精准定位”。当然也有部分作品没有标序，完全由演奏者来决定演奏顺序，这便具备了一定偶然性，是偶然音乐（或称“机遇音乐”）作曲家们极为青睐的手法之一。</span><span style=\"font-size: 16px;font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif> </span></p><p style=\"margin-bottom: 25px;line-height: 个小部分如同树叶一般散乱分布，各自具有不同色彩和生命力，每片“树叶”可能包含几个小节，一片“树叶”最后会对即将演奏的下一片“树叶”有所指示，包括速度、力度、奏法，但并不规定演奏顺序。因此演奏家可以任意选择片段进行演奏，已经选择的片段也可再次演奏，当同一片“树叶”被重复演奏三次，全曲便宣告结束。没有固定顺序的谱例注定了演奏版本会非常多变，可开始于任意部分，也可结束于任意部分，演奏家便以莫大的权力参与到作品的制作进程。</span></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">偶然音乐对图表谱的使用较多，如约翰·凯奇（John Cage）《臆想的风景第五号》（Imaginary Landscape No.5，1952），这是凯奇第一部磁带作品，其中使用了42个不同唱片，也正是从这部作品开始他才更为关注图表谱。这部作品只有一个乐章，总谱中没有音符，有的只是由“几条直线构成的（网格状）图形”，每个网格代表3英寸磁带（约0.2秒长），共有8条音轨，可在42个唱片中任意选择（同时间不超过8个）唱片进行播放。虽然时间被规定得较为精准，且每个唱片都有规定不同的持续时间和音量，但具体播放内容和顺序仍由演奏者决定，作曲家的“唯一贡献”只是告知如何使用录音带，因此仍具有相当的偶然性。</span></p><p style=\"margin-left: 8px;margin-right: 8px;\"><img class=\"rich_pages js_insertlocalimg\" data-ratio=\"0.6830409356725147\" data-s=\"300,640\" data-type=\"jpeg\" data-w=\"855\" style=\"max-width: 600px\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_jpg/c2Sib3Mp7pONQ3TtNlWUxJXeH0OoLuBuw7ia7xIkINogaeSdiaZoicOgQ2Um4WfSmoJt0ezoAzKWOiac3pLIWM3ZplA/640?wx_fmt=jpeg\"></p><p style=\"line-height: 2em;margin-bottom: 15px;margin-left: 8px;margin-right: 8px;\"><span style=\"color: rgb(0, 122, 170);font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>约翰·凯奇《臆想的风景第五号》</span><span style=\"font-size: 16px;\"></span></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">利盖蒂（György Ligeti）的管风琴曲《音量》（Volumina）也属于图表谱范畴，作曲家以图形表示音高，只能看到音的大致运动方向，音高不确定。施托克豪森的《电子音乐练习曲第二首》（Elektronische Studien II）也是如此，所谓的“谱例”中没有任何音符，仅有不同图形（三角形、方形等）和一些音频数据的坐标指示。</span></p><p style=\"text-align: center;margin-left: 8px;margin-right: 8px;\"><img class=\"rich_pages js_insertlocalimg\" data-ratio=\"0.550921435499515\" data-s=\"300,640\" data-type=\"jpeg\" data-w=\"1031\" style=\"max-width: 600px\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_jpg/c2Sib3Mp7pONQ3TtNlWUxJXeH0OoLuBuwa72k4CfOWczkB1mYzykcD79PQbQFXlVzPEXWq3ho87WiaBVSnBDGvfw/640?wx_fmt=jpeg\"></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"color: rgb(0, 122, 170);font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>利盖蒂《音量》</span><span style=\"font-size: 16px;\"></span></p><p style=\"text-align: center;margin-left: 8px;margin-right: 8px;\"><img class=\"rich_pages js_insertlocalimg\" data-ratio=\"0.46034816247582205\" data-s=\"300,640\" data-type=\"jpeg\" data-w=\"1034\" style=\"max-width: 600px\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_jpg/c2Sib3Mp7pONQ3TtNlWUxJXeH0OoLuBuwQlGKcGd0uFyaGLVwcNLdvR9wE42X9yVdfEibNJhJYqndOY64IMIqENg/640?wx_fmt=jpeg\"></p><p style=\"line-height: 2em;margin-bottom: 15px;margin-left: 8px;margin-right: 8px;\"><span style=\"color: rgb(0, 122, 170);font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>施托克豪森《电子音乐练习曲 第二首》</span><span style=\"font-size: 16px;font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif></span></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">埃尔勒·布朗（Earle Brown）《1952年12月》（December 1952）的总谱中也没有任何传统的音乐标记，只有31 个黑色长方形，若干个点，以及不同长短、粗细的线条。潘德雷茨基的《折射》（Anaklasis，1959-1960）和赫尔穆特·拉亨曼的《基罗》（Guero，1969）也属于此种类型。除了这些，还有以形体图画记录的总谱，如叶基莫夫斯基（Victor Ekimovsky）的《芭蕾》（Balletto），其总谱只是指挥动作的记录。</span></p><p style=\"text-align: center;margin-left: 8px;margin-right: 8px;\"><img class=\"rich_pages js_insertlocalimg\" data-ratio=\"0.754054054054054\" data-s=\"300,640\" data-type=\"jpeg\" data-w=\"370\" style=\"max-width: 600px\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_jpg/c2Sib3Mp7pONQ3TtNlWUxJXeH0OoLuBuw3hW0S31QiaEtqCcKw9b9I3jHs99GvEA06AE7ibNicvwooO7S4Gjo7XgGQ/640?wx_fmt=jpeg\"></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"color: rgb(0, 122, 170);font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>叶基莫夫斯基《芭蕾》</span><span style=\"font-size: 16px;\"></span></p><p style=\"text-align: center;margin-left: 8px;margin-right: 8px;\"><img class=\"rich_pages js_insertlocalimg\" data-ratio=\"0.9101654846335697\" data-s=\"300,640\" data-type=\"jpeg\" data-w=\"423\" style=\"max-width: 600px\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_jpg/c2Sib3Mp7pONQ3TtNlWUxJXeH0OoLuBuweXWp8XJVhTSozOksDibibLWumoMdCiaaOeGGQibQGbicibaA74huibFvGicFng/640?wx_fmt=jpeg\"></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"color: rgb(0, 122, 170);font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>埃尔勒 · 布朗《1952年12月》 </span><span style=\"font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif></span></p><p style=\"margin-bottom: 25px;line-height: 2em;text-align: center;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 18px;\"><strong><span style=\"font-size: 18px;background-color: rgb(0, 122, 170);color: rgb(255, 255, 255);\">时间谱</span></strong></span></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">还有一些谱式向前走得更远，比如没有音符、没有固定时值、也没有小节线的时间谱。失去小节线的作曲家大胆使用秒数表示时值，同时结合一些图表、图示、线条，以符号之间的横向距离表示音符的长短。如潘德雷茨基1961年的作品《多棱体》（Polymorphia），从手稿中我们可以看到不同颜色和形状的图案，每个图案都是对乐器演奏的指示，不同宽度代表不同的音响厚度，尖锥一般的图案便代表音响的由厚至薄。以这种方法记谱表现了多棱体的丰富变化，体现出潘德雷茨基这一时期对色彩和音响的追求。</span></p><p style=\"margin-left: 8px;margin-right: 8px;\"><img class=\"rich_pages js_insertlocalimg\" data-ratio=\"1.2293729372937294\" data-s=\"300,640\" data-type=\"jpeg\" data-w=\"606\" style=\"max-width: 600px\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_jpg/c2Sib3Mp7pONQ3TtNlWUxJXeH0OoLuBuwV3JiaYgUdg41ZYhB60M51sbS7NKmc5QXxqwWgxbsWTwsu88Ko3AJTVQ/640?wx_fmt=jpeg\"></p><p style=\"line-height: 2em;margin-bottom: 15px;margin-left: 8px;margin-right: 8px;\"><span style=\"color: rgb(0, 122, 170);font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>潘德雷茨基《多棱体》</span><span style=\"font-size: 16px;\"></span></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">同时期创作的《广岛受难者的挽歌》（Threnodyforthe Victimsof Hiroshima，1960）以及《第一弦乐四重奏》（Quartet for StringsNo.1，1960）也是使用秒数表示音响时间的时间谱范例。这两部作品都使用了图表加坐标的谱例形式，是秒数与音响的结合。图表中的横轴代表时间，以秒为单位，纵轴表示不同乐器，需根据横轴的指定时间开始或结束演奏。相比使用小节线的传统记谱，时间谱的演奏更为困难，对指挥的时间把控能力也是一种考验。</span></p><p style=\"text-align: center;margin-left: 8px;margin-right: 8px;\"><img class=\"rich_pages js_insertlocalimg\" data-ratio=\"0.584375\" data-s=\"300,640\" data-type=\"jpeg\" data-w=\"1280\" style=\"max-width: 600px\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_jpg/c2Sib3Mp7pONQ3TtNlWUxJXeH0OoLuBuwpXOK0IwW2Zibgxs7GqC2ZcznibMzwdJ4ytj2KPXzb3wtPIHfFRtBy4Rw/640?wx_fmt=jpeg\"></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"color: rgb(0, 122, 170);font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>潘德雷茨基《第一弦乐四重奏》</span><span style=\"font-size: 16px;\"></span></p><p style=\"margin-left: 8px;margin-right: 8px;\"><img class=\"rich_pages js_insertlocalimg\" data-ratio=\"0.5141843971631206\" data-s=\"300,640\" data-type=\"jpeg\" data-w=\"846\" style=\"max-width: 600px\" src=\"http://img2.jintiankansha.me/get?src=http://mmbiz.qpic.cn/mmbiz_jpg/c2Sib3Mp7pONQ3TtNlWUxJXeH0OoLuBuwwN2qHTsOFXDWUcffibpxlMKBeWr6dHOLoUTHZhgnBibA5LaeskkD1hPg/640?wx_fmt=jpeg\"></p><p style=\"line-height: 2em;margin-bottom: 15px;margin-left: 8px;margin-right: 8px;\"><span style=\"color: rgb(0, 122, 170);font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>潘德雷茨基《广岛受难者的挽歌》 </span></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">其他典型的时间谱作品如贝里奥（Luciano Berio）的《模进III》（Sequenza III，1966），该作品没有准确音高和节奏，仅要求每行乐谱的表演时长为30秒，并通过分解语言，将其变成接近乐音的无语义音节。</span><span style=\"font-size: 16px;font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif> </span></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">时间记谱法在20世纪50至60年代非常盛行，提倡以精确的时间安排获得非精确的音乐效果，与传统记谱法形成了鲜明对比。</span></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">总而言之，20世纪新出现的记谱法种类繁多，不同作曲家、技术流派可能会创用不同谱式，且新音乐作品的谱式不存在统一标准，这给新音乐及新式记谱法的理解、传播与接受造成了许多困难。</span><span style=\"font-size: 16px;font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif> </span></p><p style=\"margin-bottom: 25px;line-height: 2em;margin-left: 8px;margin-right: 8px;\"><span style=\"font-size: 16px;\">此外，由于20 世纪新音乐作品在调式调性、节奏节拍、常用乐器、组织方法等各方面都与传统古典音乐大不相同，过去惯用的赏析方法便不再起效，这种陌生感于听众与音乐之间构筑了一道天堑，想要翻越似乎极为困难。即便如今已有所改善，当下的大多数听众仍处于对其敬而远之的状态，并给它们贴上“难以理解、门槛太高、枯燥无趣、不好听”的标签。事实上，现代音乐作品并不像洪水野兽般可怕、难以接近，只要耐下心来慢慢靠近，定能领会</span><span style=\"font-size: 16px;font-family: mp-quote, -apple-system-font, BlinkMacSystemFont, \" helvetica neue sc sans gb yahei ui arial sans-serif>到许多精彩与绝妙。</span></p><p style=\"margin-bottom:
