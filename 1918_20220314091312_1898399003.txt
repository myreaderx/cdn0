powered-by=\"xiumi.us\"><p><br></p><p>2022年的冬奥会已经结束，在本次冬奥会上，中国获得了9块金牌，获得了前所未有的好成绩。<br></p><p><br></p><p>韩国也拿到了2金，维护了自己的短道速滑强国地位。不过围绕短道速滑上的竞争、判罚问题，也引发了中韩双方的口水战。</p><p><br></p><p>由于中国短道队的主教练金善台是韩国人、技术代表维克多-安（安贤洙）曾经出身于韩国，所以这两人也在韩国国内被斥之为叛徒，遭受了巨大的批评和压力。</p><p><br></p><p>3月10日，金善台终于接受了韩国媒体CBS的采访，面对镜头，金教练讲述了自己在冬奥会期间的心路历程。</p><p><br></p><p style=\"text-align: center;\"><img class=\"rich_pages wxw-img\" data-galleryid=\"\" data-ratio=\"0.6311475409836066\" data-s=\"300,640\" data-type=\"png\" data-w=\"610\" style=\"max-width: 600px\" style=\"text-align: center;\"><img class=\"rich_pages wxw-img\" data-galleryid=\"\" data-ratio=\"0.6816720257234726\" data-s=\"300,640\" data-type=\"png\" data-w=\"622\" style=\"max-width: 600px\" style=\"color: rgb(0, 0, 0);font-size: 15px;letter-spacing: 1px;\">金善台</span>说：“我尽量不去看韩国新闻，很多内容都很糟糕，我很沮丧。”</p><p><br></p><p>作为韩国队的教练，金教练带队在平昌冬奥会上获得了三块金牌，由于未能妥善处理和沈锡希的关系，金教练随后被韩国冰上联盟禁赛一年。在他不能担任任何韩国队教练期间，中国给了他一份1年期的合同。</p><p><br></p><p><span style=\"color: rgb(0, 0, 0);font-size: 15px;letter-spacing: 1px;\">金善台</span>说，“有些人认为我是在平昌冬奥会后立刻就去了中国，但是实际上我做的是跨界选项组的教练，目的是为了确认那些从直排轮滑和速滑转短道的选手的潜力。”</p><p><br></p><p>在水平得到认可后，金善台在去年10月才接过中国短道队的教鞭。</p><p><br></p><p><img class=\"rich_pages wxw-img\" data-ratio=\"0.666796875\" data-type=\"jpeg\" data-w=\"2560\" style=\"max-width: 600px\"></p><p><br></p><p>其他员工也是如此，金教练说：“除了维克多-安教练外，很多人都是和中国国家队签订了单独的合同。”</p><p><br></p><p>对于韩国国内所说，林孝俊也是<span style=\"color: rgb(0, 0, 0);font-size: 15px;letter-spacing: 1px;\">金善台</span>挖过去的说法，<span style=\"color: rgb(0, 0, 0);font-size: 15px;letter-spacing: 1px;\">金善台</span>摇头说：“林孝俊早在我担任中国主教练前，就已经加入了中国国籍。而我只有奥运会期间的合同，我怎么能够动员并负责他的归化呢。”</p><p><br></p><p>CBS问，你向中国透露了韩国短道的战术秘密吧？</p><p><br></p><p><span style=\"color: rgb(0, 0, 0);font-size: 15px;letter-spacing: 1px;\">金善台</span>说：“短道的技术水平每一年都在进步，比赛策略也是每年都在发生变化，我去年10月才开始担任中国国家队的主教练。”</p><p><br></p><p>也就是说，韩国短道这个周期的技术和战术秘密，他其实完全不了解。</p><p><br></p><p>对于韩国国家队主教练不能出国执教的观点，<span style=\"color: style=\"color: rgb(0, 0, 0);font-size: 15px;letter-spacing: style=\"text-align: center;\"><img class=\"rich_pages wxw-img\" data-galleryid=\"\" data-ratio=\"0.6816720257234726\" data-s=\"300,640\" data-type=\"png\" data-w=\"622\" style=\"max-width: 600px\"
