normal;font-family: mp-quote, -apple-system-font, system-ui, \" helvetica neue sc sans gb yahei ui arial sans-serif normal initial><span style=\"font-size: 15px;\">智商方面也是如此，已经有可靠的证据表明，两性之间的平均智商没有差别。马德里自治大学的心理学家罗伯托·科洛姆</span><span style=\"font-size: 12px;color: rgb(136, 136, 136);\">（Roberto Colom）</span><span style=\"font-size: 15px;\">经过对1989至1995年间申请私立大学的一万多名成年人的测试发现，两性“一般智商”之间的差异可以小到忽略不计</span><span style=\"font-size: 12px;color: rgb(136, 136, 136);\">（该测试将智商、认知能力和心理能力都包含在内了）</span><span style=\"font-size: 15px;\">。他的论文发表于2000年的《智慧》</span><span style=\"font-size: 12px;color: rgb(136, 136, 136);\">（Intelligence）</span><span style=\"font-size: 15px;\">杂志上，证实了此前研究中一再显示的结论。</span></p><p style=\"white-space: normal;font-family: mp-quote, -apple-system-font, system-ui, \" helvetica neue sc sans gb yahei ui arial sans-serif normal initial><br></p><p style=\"white-space: normal;font-family: mp-quote, -apple-system-font, system-ui, \" helvetica neue sc sans gb yahei ui arial sans-serif normal initial><span style=\"font-size: 15px;\">有些人认为，从统计数据来看，男性之间比女性之间有更多的变化，也就是说，尽管普通男性并不比普通女性智商高，但在该群体中有相对更多智商极低的男性和智商极高的男性。他们说，在钟形曲线的远端，也就是重叠末尾的地方，差异就变得明显。这可能就是哈佛大学校长劳伦斯·萨默斯在尝试辩解为何顶尖大学中男教授多于女教授时所提出的有争议主张的基础。</span></p><p style=\"white-space: normal;font-family: mp-quote, -apple-system-font, system-ui, \" helvetica neue sc sans gb yahei ui arial sans-serif normal initial><br></p><p style=\"white-space: normal;font-family: mp-quote, -apple-system-font, system-ui, \" helvetica neue sc sans gb yahei ui arial sans-serif normal initial><span style=\"font-size: 15px;\">科学研究并不完全支持这一主张。2008年，爱丁堡大学的一组研究人员通过对苏格兰全民范围内的11岁儿童的一般智力情况进行调查证实，男性在其测试结果中确实表现出更多的可变性。他们指出，这些差异虽然并不像某些人过去暗示的那样极端，但确实是真实存在的。同时，这些研究人员指出，最大的影响出现在量表底端。智商得分最低的人往往是男性，这在一定程度上与遗传基因有关。比如说，X连锁精神发育迟滞综合征</span><span style=\"font-size: 12px;color: rgb(136, 136, 136);\">（X-linked mental retardation）</span><span style=\"font-size: 15px;\">对男性的影响远远超过女性。</span></p><p style=\"white-space: normal;font-family: mp-quote, -apple-system-font, system-ui, \" helvetica neue sc sans gb yahei ui arial sans-serif normal initial><br></p><p style=\"white-space: normal;font-family: mp-quote, -apple-system-font, system-ui, \" helvetica neue sc sans gb yahei ui arial sans-serif normal initial><span style=\"font-size: 15px;\">海因斯也证实了这一点：“主要是因为男性有更多的发育障碍病例，所以处于最底层。在上端，差异并不大。”苏格兰的研究团队发现，他们在数据顶端看到的较小规模差异肯定不足以解释两性在数学和科学领域的差距。在他们的一组特定数据集中，男孩与女孩获得最高智商测试分数的比例大约是2:1。在大学里，男女科学教授的人数差距通常要大得多。</span></p><p style=\"white-space: normal;font-family: mp-quote, -apple-system-font, system-ui, \" helvetica neue sc sans gb yahei ui arial sans-serif normal initial><br></p><p style=\"white-space: normal;font-family: mp-quote, style=\"white-space: normal;font-family: mp-quote, -apple-system-font, system-ui, \" helvetica neue sc sans gb yahei ui arial sans-serif normal initial><br></p><p style=\"white-space: 15px;\">这一观察得到了最近研究的支持，即人们通常是如何将天才视为一种男性特质的。2015年，发表在《科学》杂志上的一项研究探讨了这种对男性天生睿智的期望是否会影响某些学科的性别平衡。在由普林斯顿大学哲学教授萨拉-简·莱斯利</span><span style=\"font-size: 12px;color: rgb(136, 136, 136);\">（Sarah-Jane Leslie）</span><span style=\"font-size: 15px;\">和伊利诺伊大学心理学家安德烈·辛皮安</span><span style=\"font-size: 12px;color: rgb(136, 136, 136);\">（Andrei Cimpian）</span><span style=\"font-size: 15px;\">主导的一项研究中，研究人员调查了来自美国三十多个学科的学者，问他们是否认为成为该领域的顶尖学者需要某种“无法传授的特殊天赋”。他们发现，在那些人们认为确实需要天赋或天才才能取得成功的学科中，女性博士的人数较少。</span></p><p style=\"white-space: normal;font-family: mp-quote, -apple-system-font, system-ui, \" helvetica neue sc sans gb yahei ui arial sans-serif normal initial><br></p><p style=\"white-space: normal;font-family: mp-quote, -apple-system-font, system-ui, \" helvetica neue sc sans gb yahei ui arial sans-serif normal initial><span style=\"font-size: 15px;\">那些努力工作会给予更高评价的学科，往往有更多女性。</span></p><p style=\"white-space: normal;font-family: mp-quote, -apple-system-font, system-ui, \"
